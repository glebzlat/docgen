Hello World library

Classes

class hello_world

Greeting generator.

This small yet super useful class allows you to print your (but not only your) name in a super cool greeting.

See also

hello world program

Public Functions

std::string get_greeting(std::string name = "World")

Function that creates greetings.

Supply you name to this function, and it will return you the string “Hello, {your-name}!”. Of course, if your PC has enough RAM to store this string, otherwise it may not work. And it also may throw an exception, and in this case you will not get your pretty greeting. Yes, life is cruel.

You even can not supply your (or any) name. In this case function will return classical “Hello, World!”. Not bad, right?

Warning

This method really can throw an exception!

Parameters:

name – Your name you want to appear in a greeting

Returns:

Instance of std::string, that contain the greeting

void print_greeting(std::string name)

Creates greeting string and prints it to stdout.

If you toooooo lazy to include <iostream>, type std::cout and std::endl and just want to get your greeting in a console, use this function. It is so kind, that it prints the greeting to stdout itself, freeing you from this boring routine. Except this feature, this function is the same as the hello_world::get_greeting function.

template<typename T>
void hello::inspect_type()

Inspect type function.

This simple function is a great help, when you’re writing highly templatized code. You can supply the type to it and see its representation.

You can do either this:

inspect_type<MyType>();

to inspect a type, or this:

inspect_type<decltype(my_var)>();

to inspect the type of a variable.
